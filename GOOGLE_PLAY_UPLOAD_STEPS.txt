═══════════════════════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════════════════════

❌ "Privacy policy URL unreachable"
   → Make sure privacy policy is publicly accessible
   → Test URL in incognito browser window

❌ "APK/Bundle upload failed"
   → Make sure file is app-release.aab not .apk
   → Check file size isn't corrupted

❌ "Some tasks incomplete"
   → Dashboard shows which tasks need completion
   → Green checkmarks = done, red X = needs work

❌ "Can't start rollout"
   → All dashboard tasks must be completed
   → Check for error messages in each section
